1. Blog posts go in `blog/_post/` folder. 

Per jekyll convention they should be named `yyyy-mm-dd-{slug-for-post}.md`  or `yyyy-mm-dd-{slug-for-post}.html` (markdown vs html)


2. Add post meta & content. 

The structure should be: 

--- 

author: {Vicki Langer} 

username: [optional] {your-user-name-if-you-have-one} 

title: {Open Innovation - A New Era} 


--- content in html or markdown goes here 

#AccessToInfo #KnowledgeSharing #SharingKnowledge #OpenData #OpenKnowledge @LeadbeaterCh #BreakingBarriers

https://www.ted.com/talks/charles_leadbeater_on_innovation?utm_campaign=tedspread&utm_medium=referral&utm_source=tedcomshare

NOTE: You need to quote any fields that contain a colon (:). For example, 

---author: {Your Name} title: 'CatchyName: Doing this thing with that thing' --- 


3. Add and commit the file then submit the pull request (if you are not pushing direct to the main repo)
